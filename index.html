Welcome to NZ Road Info

This is a personal project authored by Hamish Clulee and Matt Wynyard. NZ Road Info is not for profit and is licensed under the MIT creative commons.

Github repository

The data being displayed comes directly from the New Zealand Transport Agency. It is updated every 10 minutes.

We encourage you to check the maps while traveling and let us know if it is or isn't accurate and to send any and all suggestions on improvements to the service and quality of the information provided

Our contact page

Plans for the future

Matt is currently developing a native iOS app and Hamish is working on a hybrid web app which will provide apps for android, iOS and windows. We are interested to see the differences in performance between the native and hybrid apps.

The user experience when using the maps is also under development. Features to be added:

  • Search by address
  • Search by route
  • Filter events by type
  • Adding Auckland data/maps
